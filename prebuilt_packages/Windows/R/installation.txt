Installing the precompiled R package for Windows

0. Download the package and save it on local disk
   Note: the same package should work both with x86 x86_64
   versions of R
1. Launch R GUI
2. Ensure you have both Rcpp and Rcpp11 packages installed
   by issuing the command:
   install.packages(c("Rcpp", "Rcpp11"))
3. Select from menubar: Packages -> Install package from 
   local file
4. Navigate to the downloaded package
5. Verify that the package loads and works correctly
   by running the code from "Examples" directory in 
   IsoSpec repository on Github:

https://github.com/MatteoLacki/IsoSpec/blob/master/Examples/R/water.R
